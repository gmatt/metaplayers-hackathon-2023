# metaplayers-hackathon-2023

## Frontend

- initialize
  - ```bash
    cd frontend
    npm install
    ```
- start
  - ```bash
    npm start
    ```
  

## Backend
- prerequisites
  - Poetry (https://python-poetry.org)
  - OpenAI api key
- initialize
  - ```bash
    cd backend
    poetry install
    ```
  - Copy `.env.example` to `.env` and fill values.
- start
  - ```bash
    poetry run start
    ```
