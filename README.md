# metaplayers-hackathon-2023

## Frontend

- initialize
  - ```bash
    cd frontend
    npm install
    ```
- start
  - ```bash
    npm start
    ```
  

## Backend
- prerequisites
  - Poetry (https://python-poetry.org)
- initialize
  - ```bash
    cd backend
    poetry install
    ```
- start
  - ```bash
    poetry run uvicorn backend.api:app
    ```
